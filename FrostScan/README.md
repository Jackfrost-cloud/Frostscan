# ❄️ FrostScan – Disk Space Analyzer

Scanne ton PC et ouvre un rapport interactif dans ton navigateur.

---

## Comment l'utiliser

### Méthode rapide (double-clic)
1. Double-clique sur **FrostScan.bat**
2. Choisis un lecteur (C:\, D:\, etc.)
3. Le rapport s'ouvre dans ton navigateur

### Si Python n'est pas installé
Le .bat te propose de l'installer automatiquement.
C'est gratuit et rapide : https://www.python.org

---

## Fonctionnalités
- Scan complet d'un lecteur ou d'un dossier
- Rapport HTML interactif avec graphes
- Filtre par nom, tri par taille
- Distingue fichiers et dossiers
- Jauge d'occupation du disque
- Aucune donnée envoyée sur Internet

---

## Fichiers
- `FrostScan.bat` → double-clic pour lancer
- `frostscan.py`  → le script Python
